试了用try catch包裹起来catch不到FTPConnectionClosedException

不管是ftpClient.isConnected()还是ftpClient.isAvailable()或者ftpClient.isRemoteVerificationEnabled()的值都为True，不能判断是否连接超时。

最后采用了一种拙劣的方法catch到了FTPConnectionClosedException

try {
    //used for check connection
    ftpClient.listFiles();
} catch (Exception e) {

    Log.info(e.getMessage());

    if (!ftp.login(ftp_url,ftp_port,ftp_username,ftp_password)) {
    	Log.error("FTP reconnect and relogin failed with " + ftp_url + ":" + ftp_port + ",username: " + ftp_username + ",password: " + ftp_password);
    } else {
    	Log.info("successfully relogin for time out exception");
    }
}
